[b]CssNameFixer[/b] - маленький модуль, написанный на ассемблере, целью которого является снятие блокировки смены никнейма на Non-Steam версиях движка Source, не выходя из игры. Изначально модуль писался для CS:S, но теоретически должен заработать и на других играх.

[b]Нужно ли устанавливать[/b]:
Если у вас в корневой папке присутствует файл "rev.ini", то, скорее всего, ваш никнейм можно изменить только правкой самого файла. Если это так, то модуль можно установить.

[b]Файлы[/b]:
 * CssNameFixer.dll - "ядро", снимающее блокировку.
 * CssNameFixer.asi - инжектор, без которого библиотека не загрузится в игру. Можно использовать любые другие виды инжекта dll.

[b]Установка[/b]:
1. Распаковать файлы в папку bin, что находится в директории с файлом hl2.exe.
2. Всё. Теперь команда "name" будет работать так же, как и всегда, а ник не вернется на старый после перезапуска игры.
